📜 서버 신규 기능 업데이트 안내 (Update Note)
최근 서버에 적용된 경제 시스템 및 편의 기능에 대한 가이드입니다.

1. 💰 주식형 주류 거래소 (Dynamic Market)
이제 직접 양조한 와인과 맥주/위스키를 변동되는 시세에 맞춰 판매할 수 있습니다. 시세는 주기적으로 변동하며, 폭등하거나 폭락할 수 있습니다.

🍷 와인 거래소 (Vinery Mod)
판매 방법: 우는 흑요석(Crying Obsidian)에 와인을 들고 우클릭하면 판매됩니다.

특징: vinery 모드의 와인들을 취급합니다.

시세 확인 명령어: /marketw

시세는 30%~300% 사이에서 랜덤하게 변동됩니다.

🔥 대박(300% 이상): 판매 최적기!

▼ 떡락(50% 미만): 존버가 필요합니다.

🍺 양조장 거래소 (Brewery Mod)
판매 방법: 와인과 동일하게 우는 흑요석에 맥주/위스키를 들고 우클릭.

특징: brewery 모드의 맥주와 위스키를 취급합니다. 와인 시세와는 별개로 움직입니다.

시세 확인 명령어: /marketb

보상: 판매 대금은 Clutter 코인 (금/은/동)으로 즉시 지급됩니다.

2. 🎫 특수 이동권 (Teleport Tickets)
명령어를 직접 입력하는 대신, 아이템을 사용하여 직관적으로 이동할 수 있는 기능이 추가되었습니다.

🏠 스폰 귀환권 (Spawn Ticket)
아이템: 특수 태그가 부여된 종이 아이템

사용법: 아이템을 들고 우클릭

효과: 마을 스폰 지점(/spawn)으로 즉시 이동합니다.

주의: 사용 시 아이템이 1개 소모됩니다.

✨ 사망 귀환권 (Back Ticket)
아이템: 특수 태그가 부여된 종이 아이템

사용법: 아이템을 들고 우클릭

효과: 가장 마지막에 사망했던 지점(/back)으로 이동합니다. 아이템을 찾으러 갈 때 유용합니다.

주의: 사용 시 아이템이 1개 소모됩니다.

3. 🌀 차원 이동 구역 (Portal Zones)
특정 좌표에 도달하면 자동으로 다른 위치로 텔레포트되는 상시 포탈 구역이 활성화되었습니다.

작동 방식: 해당 구역에 서 있으면 약 1초 뒤 자동으로 이동됩니다.

이동 경로:

좌표 x: -468, z: 1676 부근 ➡ x: -241, z: -769 로 이동

좌표 x: -241, z: -744 부근 ➡ x: -464, z: 1676 로 이동

알림: 이동 시 "🌀 차원 이동 구역을 통과했습니다!" 메시지와 함께 사운드가 재생됩니다.
